# Prescripto – Doctor Appointment Booking App

## Project Description
Prescripto is a full-stack doctor appointment booking application that allows users to book appointments, doctors to manage schedules, and admins to control the system.  

It also includes a **telemedicine video call feature for doctor-to-doctor communication**, implemented using **Socket.IO** for real-time signaling.

## Features
- User registration and login
- Doctor appointment booking
- Admin dashboard
- Doctor management
- Secure authentication
- Telemedicine video call between doctors (real-time)
- Real-time communication using Socket.IO


## Tech Stack
- Frontend: React
- Admin Panel: React
- Backend: Node.js, Express
- Database: MongoDB
- Real-time Communication: Socket.IO
- Video Call:Socket.IO
